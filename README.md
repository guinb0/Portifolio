# Guilherme Nunes Bezerra — Portfolio

Full Stack Developer portfolio built with Flask.

**Live site:** https://guinb0.github.io/Portifolio/

> First-time setup: in the repo go to **Settings → Pages → Build and deployment**, set **Source** to **Deploy from a branch**, branch **`gh-pages`** and folder **`/ (root)`**, then save. The site goes live in 1–2 minutes.

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000

## Deploy

Pushes to `main` automatically build and deploy to GitHub Pages via `.github/workflows/deploy-pages.yml`.
