# Bandeiras para Seleção de Idioma

Coloque as seguintes imagens nesta pasta:

- `brazil.png` - Bandeira do Brasil
- `usa.png` - Bandeira dos Estados Unidos
- `spain.png` - Bandeira da Espanha

## Recomendações:
- Formato: PNG com fundo transparente
- Tamanho recomendado: 120x120px ou maior (será redimensionado automaticamente)
- Qualidade: Alta resolução para telas retina

## Onde encontrar bandeiras:
- [Flaticon](https://www.flaticon.com/search?word=flags)
- [Freepik](https://www.freepik.com/search?format=search&query=country%20flags)
- [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:National_flags)
