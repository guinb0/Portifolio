# Guilherme Nunes Bezerra — Portfolio

Full Stack Developer portfolio built with Flask.

**Live site:** https://guinb0.github.io/Portifolio/

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000

## Deploy

Pushes to `main` automatically build and deploy to GitHub Pages via `.github/workflows/deploy-pages.yml`.
